# OnAndHome 2차 — 프론트엔드

## 프로젝트 개요
Spring Boot + React 기반 온라인 쇼핑몰 개선 프로젝트 프론트엔드.
Thymeleaf 정적 구조를 React 컴포넌트 아키텍처로 전환하고 핵심 기능 고도화.

## 관련 레포
- 백엔드: [OnAndHomeBack](https://github.com/roach-hue/OnAndHomeBack)

## 팀 구성 및 담당
- 팀원 6명
- 본인 담당: 프론트엔드 전담
  - Thymeleaf 정적 마크업 → React 컴포넌트 기반 아키텍처 전환
  - 별점 리뷰 + 멀티 이미지 첨부 기능 구현
  - 관리자 대시보드 재설계
  - 카테고리별 상세 페이지 레이아웃 다각화

## 기술 스택
React 19 · Redux Toolkit · React Router · Axios · WebSocket (STOMP) · Tailwind CSS
